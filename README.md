# Snakemake workflow: QCrs4

This workflow performs basic QC analysis for NGS data.\
QCrs4 is part of the Snakemake-based pipelines collection [solida-core](https://github.com/solida-core) developed and manteined at [CRS4](https://www.crs4.it).
